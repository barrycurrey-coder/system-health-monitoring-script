# Linux System Health Monitoring Script

Hi, I'm Barry, a student learning Linux and system administration.

This project demonstrates my ability to automate system monitoring using Bash scripting in a Linux environment.

---

## 🚨 Problem
System administrators often need to manually run multiple commands to check system health (CPU, memory, disk usage, and processes). This process is repetitive, time-consuming, and prone to human error.

---

## ✅ Solution
This script automates system health checks and generates a clean, readable report that includes:
- CPU usage
- Memory usage
- Disk usage
- Running processes

By consolidating this information into a single script, it improves efficiency and helps quickly identify potential system issues.

---

## 🛠️ Technologies Used
- Bash scripting  
- Linux commands (`ps`, `df`, `free`, `uptime`, `awk`, `grep`)  
- GitHub Codespaces
